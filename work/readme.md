# Homework 12 - Employee Tracker

## Minimum Requirements

- Functional application ✅&nbsp;

- GitHub repository with a unique name and a README describing the project ✅&nbsp;

### Command-line interface allows user to:

- Add employees by role ✅&nbsp;

- View departments, roles, employees ✅&nbsp;

- Update employee roles ✅&nbsp;

## Bonus

### The command-line application should allow users to:

- Update employee managers

- View employees by manager

- Delete:

  - Departments
  - Roles
  - Employees ✅&nbsp;

- View the total utilized budget of a department -- ie the combined salaries of all employees in that department

## Usage

### Installation

1. Run `npm install`
2. Type `node server.js`

### Navigation

Use the arrow keys and enter to navigate through directory
